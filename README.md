# Wally Documentation

## Home Page
The Home Page Showcases key features and benefits of the app.
- Highlights financial management capabilities.
- Displays user testimonials and success stories.
- Provides pricing information and subscription tiers.
- Clear call-to-action buttons to sign up or learn more.
- Navigation links to different sections of the website.


![Home Page Screenshot](https://github.com/user-attachments/assets/f2082a46-dfb2-4af8-854b-81072a5635a7)



## Dashboard
The Dashboard is the main interface where users can view a summary of their financial data. Key features include:
- Overview of account balances
- Visual charts representing spending trends
- Quick access to recent transactions


![Dashboard Screenshot](https://github.com/user-attachments/assets/7ff927ad-0fda-40d8-b3d6-6aabd5578887)



## Transactions
The Transactions page allows users to view and manage all transactions.
- Users can add, edit, or delete transactions.
- Search functionality to find specific transactions.


![Transactions Screenshot](https://github.com/user-attachments/assets/cd3f5fa2-54e6-4502-b856-49c798f49822)


## Analytics
The Analytics section provides detailed insights into user spending patterns.
- Users can filter expenses by category, date range, and account.
- Visual reports help in understanding expenditure habits.


![Analytics Screenshot](https://github.com/user-attachments/assets/c6f54711-bc41-4231-8816-7b2fe8edbcbd)


## Budgets
In the Budgets section, users can set monthly budgets for different categories.
- Notifications when spending approaches the budget limits.
- Historical budget analysis.


![Budgets Screenshot](https://github.com/user-attachments/assets/67de9011-d5d1-4f49-aad1-5d3f656e9d44)



## Notifications
This section is dedicated to alerts and reminders related to transactions and budgets.
- Users can customize notification settings for different events.

![Notifications Screenshot](https://github.com/user-attachments/assets/450689f3-685c-4bd9-a10b-a8a1dc702e27)
  

## Auth
The Authentication section handles user logins and security settings.
- Supports multi-factor authentication (MFA).
- Password recovery options for users.


![Authentication Screenshot](https://github.com/user-attachments/assets/f147d073-1779-41ea-9da2-7c8a0d93c1a1)


## Settings
This section is dedicated to control user's prefrences like: 
- Update profile information and wallet name
- Change currency preference (USD, OMR, EUR, SAR)
- Select language preference for the interface(set as coming soon)
- Manage account security and logout functionality
